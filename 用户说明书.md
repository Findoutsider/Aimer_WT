# Aimer WT 语音包管理工具 - 用户说明书

## 📖 软件简介

Aimer WT 是一款专为《战争雷霆》玩家打造的语音包管理工具，可以帮助您轻松地安装、管理和切换各类自定义语音包。

**版本**: v1 Beta  
**作者**: Aimer

## 🚀 快速开始

### 第一步：设置游戏路径

1. 启动软件后，首先需要设置游戏安装路径
2. **自动搜索**：点击主页的"自动搜索"按钮，软件会自动扫描您的硬盘寻找游戏目录
3. **手动选择**：点击"浏览"按钮，手动选择《战争雷霆》的安装目录
4. 正确的路径应包含 `config.blk` 文件（例如：`D:\Games\War Thunder`）

### 第二步：导入语音包

1. 切换到"语音包库"页面
2. 点击左上角"导入 ZIP"按钮
3. 选择一种导入方式：
   - **选择文件导入**：浏览并选择任意位置的 ZIP 压缩包
   - **一键批量导入**：自动扫描软件目录下的 `WT待解压区` 文件夹中的所有 ZIP 文件

### 第三步：安装语音包

1. 在语音包库中找到您想使用的语音包
2. 点击卡片右侧的播放按钮（▶）
3. 在弹出的窗口中选择要安装的模块（如：陆战语音、无线电等）
4. 点击"立即加载"按钮

## 🎮 功能说明

### 主页功能
- **游戏路径设置**：配置游戏安装目录
- **连接状态**：显示当前是否成功连接到游戏目录
- **运行日志**：实时显示软件的操作日志

### 语音包库
- **导入 ZIP**：导入新的语音包压缩文件
- **库目录**：打开语音包存储文件夹
- **待解压区**：打开待导入文件夹
- **游戏目录**：打开游戏安装目录
- **搜索**：支持按标题或作者名搜索
- **删除**：移除不需要的语音包

### 设置页面
- **资源管理**：快速打开各类文件夹
- **个性化**：切换界面主题风格
- **一键还原**：清空所有已安装的语音包，恢复游戏默认状态

## ❓ 常见问题

### Q: 语音包安装后游戏内没有效果？
A: 请检查以下几点：
1. 确保游戏目录设置正确
2. 检查 `config.blk` 文件中是否有 `enable_mod:b=yes`
3. 重启游戏后再试
4. 如问题持续，请联系语音包制作者

### Q: 如何卸载语音包？
A: 在设置页面点击"一键还原纯净模式"按钮，会清空游戏目录下的 `sound/mod` 文件夹并禁用 mod 配置。

### Q: 可以同时安装多个语音包吗？
A: 可以，但同名文件会被覆盖。建议一次只安装一个语音包的相同类型模块。

### Q: 软件打包成 exe 后无法正常工作？
A: v1 Beta 版本已修复此问题。如果仍有问题，请确保使用最新版本的软件。
